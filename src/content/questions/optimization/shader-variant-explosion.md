## 参考答案

### 🎬 场景描述

面试官说：「我们项目升级了 URP，加了大批 multi_compile 关键字后，构建出的安卓包里 Shader 变体数量飙升到 3 万多个，变体编译占了一半构建时间，APK 也大了 200MB。你作为 TA 怎么排查和瘦身？」

### ✅ 核心要点

1. **变体来源审计**：搞清楚 3 万个变体是哪些 Shader / 哪些关键字排列组合出来的
2. **关键字裁剪**：去掉用不到的关键字组合（`#pragma multi_compile` → `#pragma shader_feature`）
3. **Shader Variant Collection**：只保留实际使用的变体，打 Strip
4. **IPreprocessShaders 接口**：编写自动化剔除脚本，构建时拦截无用变体
5. **关键字组合数学**：理解 `multi_compile` 是乘法关系，减少 1 个关键字 = 变体减半

### 📖 深度展开

#### 解决思路（从效果倒推实现）

```
问题现象：3 万个变体 → 构建慢 + 包体大 + 运行时内存高
                ↑
倒推1：变体从哪来？→ 导出变体列表，按 Shader 分组统计
倒推2：哪些关键字贡献最多？→ 每个 multi_compile 有 N 个选项，M 个组合 = N^M
倒推3：哪些组合真的用到？→ 扫描场景/预制体中的材质关键字 → 生成 SVC
倒推4：哪些关键字可以改？→ multi_compile（全保留）vs shader_feature（按需）
倒推5：构建时怎么拦截？→ IPreprocessShaders 回调剔除不在 SVC 中的变体
```

#### 代码实现

**1. 变体统计工具（编辑器脚本）：**

```csharp
using System.Collections.Generic;
using System.Linq;
using UnityEditor;
using UnityEngine;

public class ShaderVariantAnalyzer : EditorWindow
{
    [MenuItem("Tools/TA/Shader Variant Analyzer")]
    static void Open() => GetWindow<ShaderVariantAnalyzer>();

    void OnGUI()
    {
        if (GUILayout.Button("Analyze All Shaders"))
        {
            AnalyzeAllShaders();
        }
        if (GUILayout.Button("Analyze URP Built-in Keywords"))
        {
            AnalyzeURPKeywords();
        }
    }

    static void AnalyzeAllShaders()
    {
        var allShaders = ShaderUtil.GetAllShaders();
        var report = new List<string>();
        long totalVariants = 0;

        foreach (Shader shader in allShaders)
        {
            int count = ShaderUtil.GetVariantCount(shader);
            if (count > 100) // 只报告变体数 > 100 的
            {
                report.Add($"[ {count,6} ] {shader.name}");
                totalVariants += count;
            }
        }

        report.Insert(0, $"=== Total Variants (filtered > 100): {totalVariants} ===\n");
        Debug.Log(string.Join("\n", report));

        // 写入文件
        System.IO.File.WriteAllText(
            "Assets/ShaderVariantReport.txt",
            string.Join("\n", report)
        );
        Debug.Log("Report saved to Assets/ShaderVariantReport.txt");
    }

    static void AnalyzeURPKeywords()
    {
        // URP 常见的 multi_compile 关键字
        string[] urpKeywords = new string[]
        {
            "_MAIN_LIGHT_SHADOWS",
            "_MAIN_LIGHT_SHADOWS_CASCADE",
            "_ADDITIONAL_LIGHTS",
            "_ADDITIONAL_LIGHT_SHADOWS",
            "_SHADOWS_SOFT",
            "_SCREEN_SPACE_OCCLUSION",
            "_EMISSION",
            "_FOG",
            "_LIGHT_COOKIES",
            "_CLUSTERED_RENDERING",
        };

        // 二进制关键字的变体数 = 2^N
        int binaryVariantCount = (int)Mathf.Pow(2, urpKeywords.Length);
        Debug.Log($"URP Binary Keywords: {urpKeywords.Length} → " +
                  $"Theoretical max variants: {binaryVariantCount} " +
                  $"({urpKeywords.Length} 个关键字的乘积)");
    }
}
```

**2. Shader Variant Collection 自动生成：**

```csharp
using System.Collections.Generic;
using UnityEditor;
using UnityEngine;

public class SVCAutoGenerator : MonoBehaviour
{
    [MenuItem("Tools/TA/Generate SVC from Scenes")]
    static void GenerateSVC()
    {
        var svc = ScriptableObject.CreateInstance<ShaderVariantCollection>();
        svc.name = "AutoGeneratedSVC";

        // 扫描所有场景中的材质，收集实际使用的变体
        string[] scenePaths = EditorBuildSettings.scenes
            .Where(s => s.enabled)
            .Select(s => s.path)
            .ToArray();

        var usedVariants = new HashSet<string>();
        int collected = 0;

        foreach (string scenePath in scenePaths)
        {
            var scene = EditorSceneManager.OpenScene(scenePath);
            foreach (var renderer in FindObjectsByType<Renderer>(FindObjectsSortMode.None))
            {
                foreach (var mat in renderer.sharedMaterials)
                {
                    if (mat == null || mat.shader == null) continue;

                    string shaderName = mat.shader.name;
                    string[] keywords = mat.shaderKeywords;
                    string key = $"{shaderName}|{string.Join(",", keywords)}";

                    if (usedVariants.Add(key))
                    {
                        var entry = new ShaderVariantCollection.ShaderVariant(
                            mat.shader,
                            PassType.Normal,
                            keywords
                        );
                        // 注意：Add 可能失败（变体不存在），需要 try-catch
                        svc.Add(entry);
                        collected++;
                    }
                }
            }
        }

        AssetDatabase.CreateAsset(svc, "Assets/AutoGeneratedSVC.asset");
        AssetDatabase.SaveAssets();
        Debug.Log($"SVC generated: {collected} variants collected from {scenePaths.Length} scenes.");
    }
}
```

**3. IPreprocessShaders 构建时剔除：**

```csharp
using System.Collections.Generic;
using UnityEditor;
using UnityEditor.Build;
using UnityEditor.Build.Reporting;
using UnityEngine;

public class ShaderVariantStripper : IPreprocessShaders
{
    private HashSet<string> _allowedVariants;
    public int callbackOrder => 0;

    public void OnPreprocessBuild(BuildReport report)
    {
        // 加载白名单 SVC
        var svc = AssetDatabase.LoadAssetAtPath<ShaderVariantCollection>(
            "Assets/AutoGeneratedSVC.asset"
        );

        _allowedVariants = new HashSet<string>();
        if (svc != null)
        {
            foreach (var entry in svc.GetEnumerator())
            {
                string key = $"{entry.shader.name}|{string.Join(",", entry.keywords)}";
                _allowedVariants.Add(key);
            }
        }

        Debug.Log($"[ShaderVariantStripper] Allowed variants: {_allowedVariants.Count}");
    }

    public void OnProcessShader(
        Shader shader,
        ShaderSnippetData snippet,
        IList<ShaderCompilerData> data)
    {
        // URP 核心 Shader 不剔除（防止出问题）
        if (shader.name.Contains("Universal Render Pipeline") &&
            !shader.name.Contains("Custom"))
            return;

        int beforeCount = data.Count;
        string shaderName = shader.name;

        for (int i = data.Count - 1; i >= 0; i--)
        {
            var keywords = data[i].shaderKeywordSet.GetShaderKeywords();
            var keywordNames = keywords.Select(k => k.GetKeywordName().ToString());
            string key = $"{shaderName}|{string.Join(",", keywordNames)}";

            if (!_allowedVariants.Contains(key))
            {
                data.RemoveAt(i);
            }
        }

        if (beforeCount != data.Count)
        {
            Debug.Log($"[Stripper] {shaderName}: {beforeCount} → {data.Count} " +
                      $"(stripped {beforeCount - data.Count})");
        }
    }
}
```

**变体来源拆解（典型 URP 项目）：**

| 来源 | 关键字数 | 理论变体数 | 说明 |
|------|---------|-----------|------|
| URP Lit.shader | ~12 | ~4,096 | 最大变体来源 |
| URP Lit (Pass: ShadowCaster) | ~8 | ~256 | 阴影 Pass |
| URP Lit (Pass: DepthOnly) | ~6 | ~64 | 深度 Prepass |
| 项目自定义 Shader A | ~5 | ~32 | 如果用 multi_compile |
| 项目自定义 Shader B | ~4 | ~16 | shader_feature 会少很多 |
| URP 后处理 | ~10 | ~1,024 | Bloom / DOF / Motion Blur 等 |
| **总计（乘法叠加）** | — | **~3 万+** | 不裁剪的真实数字 |

### ⚡ 实战经验

- **先抓大头**：80% 的变体来自 20% 的 Shader。URP 自带的 Lit.shader 通常是变体之王，优先处理
- **URP Asset 关键字裁剪**：Project Settings → Graphics → Built-in Shader Stripping 里可以按需关闭 URP 功能关键字
- **shader_feature 陷阱**：多个 `shader_feature` 组合在一起，如果不做材质预处理，变体数和 `multi_compile` 一样爆炸。关键在于材质上是否真正启用了这些关键字
- **运行时 SVC 记录**：`ShaderVariantCollection` 可以在运行时 `RecordVariants()`，在真机测试时录一段完整流程，导出的 SVC 最准确
- **SVC 不完整的风险**：如果 SVC 里漏了一个变体，运行时切换到该变体时会卡顿（JIT 编译）。一定要覆盖所有角色/场景/天气切换

| 卡点 | 盲区 | 补习方向 |

# 技术美术（TA）面试指南

🎨 面向技术美术岗位的结构化面试题库 — Shader · 渲染管线 · 性能优化 · 工具链 · PBR

## 技术栈

- **Astro** + **React** — 静态站点生成
- **Pagefind** — 全文搜索
- **GitHub Pages** — 自动部署

## 分类

| 分类 | 说明 |
|------|------|
| 🎨 技术美术基础 | TA 岗位认知、工作流、跨部门协作 |
| 🌈 Shader 编写 | HLSL/GLSL、Shader Graph、顶点/片元着色器 |
| ⚡ 渲染管线 | Forward/Deferred/Forward+、PBR、光照模型 |
| 🚀 性能优化 | Draw Call、Overdraw、GPU Profiling、LOD |
| 🔧 工具与管线 | Houdini、Maya/Blender、自动化、资产规范 |
| 🟢 非技术面 | 团队协作、项目经验、作品集展示 |
